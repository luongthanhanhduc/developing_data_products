Getting and Cleaning Data

The dataset was collected using the package WDI. There are 5 indicators we collected for this dataset:

  1. GDP per capita (current US dollar)
  2. GDP growth (as percentage of GDP in current US dollar)
  3. GNI per capita (current US dollar)
  4. Foreign direct investment (current US dollar)
  5. total population

Summary of the dataset

Details about how we collected and clean the data can be found from the documentation in the website

Visualization with World Map

Visualization of GDP per capita (current US dollar) at 2010

## Creating a generic function for 'toJSON' from package 'jsonlite' in package 'googleVis'

map <- gvisGeoChart(country_data[country_data$year == 2010,], locationvar="country", 
                    colorvar="GDP per capita", options=list(projection="kavrayskiy-vii"))
print(map, "chart")

Data visualization for specific country

Visualization of GDP growth of Vietnam

chart <- gvisColumnChart(data = country_data[country_data$country == "Vietnam", ], 
                         xvar = "year", yvar = "GDP growth")
print(chart, "chart")
